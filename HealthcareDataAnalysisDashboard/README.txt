# Power BI Project: Healthcare Data Analysis

This dashboard analyzes hospital-level data across New York State, focusing on discharges, cost per discharge, average length of stay, and diagnosis-based segmentation.

## Features
- KPI summary cards
- Interactive slicers (region, diagnosis, risk)
- Top/Bottom 3 insights
- Diagnosis and age group analysis
- Cost and LOS breakdowns by hospital

## Best Practices Followed
- Clean star schema and data modeling
- Descriptive column naming
- Logical layout and visual hierarchy
- Minimal clutter with focused insights per page
- Performance-optimized DAX and visuals
- Accessibility through labels and tooltips

## Tools Used
- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
